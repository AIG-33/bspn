# Закрытая Telegram-группа для членов союза

Пошаговая инструкция: создание закрытой группы с темами и запуск бота,
который делает красивое закреплённое меню и позволяет админам добавлять темы.

## Важно знать

Сам Telegram не позволяет ботам **создавать группы** — группу создаёт человек
(1 минута), а дальше всё делает бот: создаёт темы, публикует и закрепляет
меню-навигацию, следит за новыми темами.

## Шаг 1. Создать бота (один раз)

1. Откройте [@BotFather](https://t.me/BotFather) → `/newbot`
2. Имя: например `БСПН Группа`, username: например `bspn_group_bot`
3. Скопируйте токен и добавьте в `.env.local`:

```
TELEGRAM_BOT_TOKEN=123456:ABC-...
```

## Шаг 2. Создать закрытую группу

1. Telegram → Новая группа → название, например **«БСПН — члены союза»**
2. Настройки группы:
   - **Тип группы: Частная** (доступ только по инвайт-ссылке)
   - **Темы (Topics): Включить** — это обязательно
3. Добавьте бота в группу и назначьте его **администратором** с правами:
   - Управление темами (Manage Topics)
   - Закрепление сообщений (Pin Messages)
   - Приглашение пользователей (Invite Users via Link) — для ссылки с заявками
   - Добавление участников / одобрение заявок (Add Members) — чтобы кнопки
     «Принять / Отклонить» работали

## Шаг 3. Запустить бота и настроить группу

```bash
npm run bot
```

В группе (от имени админа) отправьте:

```
/setup
```

Бот создаст стартовые темы и закрепит меню с кнопками-ссылками на каждую тему:

- 📰 Новости
- 💬 Обсуждения
- 🎬 Контент для медиа

## Как админы добавляют темы

Два способа — оба автоматически обновляют закреплённое меню:

1. Командой в любой теме:

```
/newtopic 🎯 Мероприятия
```

2. Обычным способом через интерфейс Telegram (⋮ → Create Topic) —
   бот заметит новую тему и добавит её в меню сам. Переименования тоже
   подхватываются.

## Команды бота

| Команда | Кто | Что делает |
| --- | --- | --- |
| `/setup` | админы | создать стартовые темы + закрепить меню |
| `/newtopic 🎯 Название` | админы | добавить тему |
| `/menu` | админы | обновить/переопубликовать меню |
| `/invite` | админы | ссылка-приглашение (вступление по одобрению) |
| `/help` | все | справка |

## Приглашение членов союза — только с одобрения админа

Команда `/invite` в группе:

- создаёт постоянную ссылку **с заявками на вступление** — по ней нельзя
  войти без одобрения администратора;
- отзывает старую основную ссылку (по которой можно было войти свободно);
- при каждой новой заявке бот публикует в группе карточку с кнопками
  **«✅ Принять / ❌ Отклонить»** — нажать может только админ. Заявки также
  видны стандартно в шапке группы.

Ссылку из `/invite` можно рассылать членам через e-mail-дайджест или лично.

## Постоянная работа бота

Бот нужен запущенным только когда админы управляют темами (меню и созданные
темы никуда не денутся, если бот офлайн). Для постоянной работы можно
запустить его на любом сервере/VPS:

```bash
TELEGRAM_BOT_TOKEN=... node scripts/telegram-group-bot.mjs
```

Состояние (id закреплённого меню и список тем) хранится в
`scripts/.telegram-bot-state.json` (не коммитится).
